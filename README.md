# Context-Aware Adaptive Patient Monitoring and Risk Escalation System

This project is a command-line based intelligent healthcare system that analyzes patient health data using machine learning and rule-based logic. It predicts risk levels, detects condition escalation, and provides appropriate medical recommendations.

## Features
- Machine learning based risk prediction
- Rule-based decision system
- Risk scoring mechanism
- Emergency detection and escalation
- Clean CLI interface

## Technologies Used
- Python
- scikit-learn
- pandas

## How to Run

1. Install dependencies:
   - pip install pandas scikit-learn
2. Run model training:
   - python model.py
3. Run the system:
   - python main.py

## Project Structure
- data_generator.py → dataset creation
- model.py → ML model training
- predict.py → prediction logic
- decision_logic.py → rule-based AI logic
- main.py → main execution file

## Output
The system provides:
- Risk Level
- Risk Score
- Condition Analysis
- Recommendation
