# Project Statement

## Problem Statement
Patients often face difficulty in understanding the seriousness of their health condition based on symptoms and vital signs. There is a need for a system that can assist in early risk detection and decision making.

## Solution
This project presents a context-aware adaptive patient monitoring system that analyzes patient data using a hybrid approach combining machine learning and rule-based logic.

## Approach
The system uses a Decision Tree classifier to predict risk levels based on patient features. A rule-based decision system further analyzes the condition to detect escalation and provide recommendations.

## Key Features
- Risk prediction using ML
- Risk scoring system
- Escalation detection
- Recommendation generation

## AI and ML Usage
Machine learning is used for classification of patient risk levels. Rule-based logic is used for intelligent decision making and escalation detection.

## Conclusion
The system demonstrates how AI and ML can be combined to build an intelligent healthcare decision support system.
